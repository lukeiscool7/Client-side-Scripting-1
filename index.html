What is hypertext?

HTML, or Hypertext Markup Language, is the fundamental language used to create web pages. It serves as the backbone, defining the structure and content of a webpage through various elements like headings, paragraphs, images, links, and more. These elements are encapsulated within tags, which instruct web browsers on how to render the content for users. HTML ensures consistency in the presentation of web content across different devices and platforms, forming the basis for the entire World Wide Web.

Benifits to learning how to use HTML(Hypertext)

  • Basic Web Skills: HTML is essential for creating web pages.
  • Understanding Web Content: Learning HTML helps understand how web content is structured.
  • Improved Collaboration: Knowing HTML enables better communication with web developers and designers.

Define internet backbone in your own words:

The internet backbone communication lines that interconnect various networks across the globe forming the infrastructure for the internet.

What is the difference between clients and servers on the internet?

Clients are devices or software applications that request and receive services or resources from servers. Servers, on the other hand, are computers or systems that provide services or resources to clients over a network.

Draw or describe how servers, routers, switches, workstations, and other hardware components are part of the internet:

Servers store and deliver data, routers direct traffic, switches connect devices, and workstations access information, collectively forming the infrastructure of the internet.

Describe the Internet Protocol Suite (TCP/IP) model of networking:

The TCP/IP model is a conceptual framework used for understanding and standardizing communication protocols on the internet. It consists of layers, each responsible for different aspects of data transmission, ensuring reliable and efficient communication between devices.

List at least three examples of internet services that rely on client/server relationships:

  • Email (SMTP/IMAP)
  • Web browsing (HTTP/HTTPS)
  • File transfer (FTP)

How is “the web” related to the internet?

The web is a subset of the internet. It is a collection of interconnected documents and resources accessible via the internet linked together through hyperlinks.

Explain how the Hypertext Transfer Protocol (HTTP) is used to facilitate client/server interaction:

HTTP is a protocol used for communication between web clients (such as browsers) and servers. It defines how messages are formatted and transmitted, allowing clients to request resources (such as web pages) from servers, which then respond with the requested content.

What is HTTP and how is it different from HTTPS?

HTTP is a protocol used for transmitting data over the internet. HTTPS is a secure version of HTTP that encrypts data sent between the client and server, providing a higher level of security.

Describe and illustrate the development of the Web in general terms using the Web 1.0, Web 2.0, and Web 3.0 generational paradigms:

The development of the web from Web 1.0 to Web 3.0 represents a progression from static, one-way information sharing to dynamic, interactive, and intelligent online experiences.

Compare and contrast the Web 1.0, Web 2.0, and Web 3.0 stages of development:

Web 1.0 was about passive consumption, Web 2.0 added interaction and user-generated content, and Web 3.0 emphasizes intelligence, decentralization, and security.

Describe the role markup languages play in the Web:

Markup languages, such as HTML and XML, are used to structure and format content on the web. They define the presentation and organization of text, images, and other media, making web content readable and accessible to users.

Compare and contrast markup and scripting languages:

Markup languages, like HTML, define the structure and presentation of content. Scripting languages, such as JavaScript, add functionality and interactivity to web pages by allowing for dynamic behavior and user interactions.

Explain the relationship between Standard Generalized Markup Language (SGML), Extensible Markup Language (XML), Hypertext Markup Language (HTML), Extensible Hypertext Markup Language (XHTML), Dynamic Hypertext Markup Language (DHTML), and markdown (MD):

SGML laid the groundwork for HTML and XML, which influenced XHTML and DHTML, while Markdown provides a simplified syntax for formatting text.
